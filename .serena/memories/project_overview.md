reocli is a Rust 2024 project providing a Reolink CLI (`reocli`) and an MCP-style command runner (`reocli-mcp`). The codebase is organized into core domain types (`src/core`), shared application usecases (`src/app`), Reolink/ONVIF adapters (`src/reolink`), interface adapters for CLI and MCP (`src/interfaces`), and binary entrypoints in `src/bin`. Main features include auth, device/time/network queries, snapshot capture, PTZ control, PTZ auto-calibration, and raw-count absolute PTZ control with CGI and ONVIF transport support.
